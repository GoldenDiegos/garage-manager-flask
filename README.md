Garage Manager – Flask + SQLite

Garage Manager es una aplicación web desarrollada con Flask (Python) y SQLite que permite gestionar usuarios, vehículos, servicios y documentos asociados a cada automóvil. El sistema implementa operaciones CRUD completas, relaciones uno a muchos, integridad referencial y una arquitectura organizada adecuada para fines académicos.

Objetivo del Proyecto

El objetivo principal del proyecto es desarrollar un sistema web funcional que permita registrar usuarios (propietarios), asociar múltiples vehículos a cada usuario, llevar un historial de servicios por vehículo y administrar documentos asociados a cada automóvil. Además, se busca aplicar correctamente conceptos de bases de datos relacionales, rutas dinámicas en Flask y buenas prácticas de seguridad en consultas SQL.
